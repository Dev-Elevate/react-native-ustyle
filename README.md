# react-native-ustyle

Easy and configurable utiltity styling props support for React Native

## Installation

```sh
npm install react-native-ustyle
```

## Usage

```js
import { multiply } from 'react-native-ustyle';

// ...

const result = await multiply(3, 7);
```
